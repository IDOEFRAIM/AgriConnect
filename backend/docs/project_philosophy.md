# Philosophie du Projet AgConnect : "Le Grand Marché National"

Basé sur les retours terrain (conversation avec un agriculteur), voici les piliers fondamentaux du projet pour répondre aux vrais problèmes du monde rural sahélien.

## 1. Le Problème Réel : L'Opacité et la Spéculation
- **Le Constat** : Les agriculteurs ne maîtrisent pas le marché. Ils ont l'impression que "quelqu'un dans l'ombre décide".
- **L'Exemple** : Le sésame passe de 1750 FCFA à 800 FCFA sans explication claire.
- **La Conséquence** : Sentiment d'impuissance, pertes financières énormes, incapacité à prévoir.
- **L'Ennemi** : Les intermédiaires spéculateurs qui profitent du manque d'information ("des gens qui trafiquent").

## 2. La Solution : Transparence et Accès Direct
- **Le Grand Marché National** : Une plateforme centralisée pour connecter directement l'offre et la demande, contournant les spéculateurs locaux.
- **L'Information comme Arme** :
    - Ne pas juste donner le prix, mais **EXPLIQUER** pourquoi il varie (Offre/Demande, Saisonnalité, Cours mondial).
    - Comparer le prix du marché local avec le prix officiel (SONAGESS) pour détecter les abus.
- **Le Rôle du "Connecteur"** : Dans les zones sans internet, un jeune connecté centralise les offres/demandes pour le village via WhatsApp/SMS.

## 3. Modèle Économique : "Business à l'Africaine" (Pay-as-you-go)
- **Pas d'Abonnement Mensuel** : Les revenus sont saisonniers et incertains. Un coût fixe est un frein psychologique et financier.
- **Paiement à l'Usage** :
    - Une petite commission sur la vente réussie (ex: 10 FCFA sur 10000 FCFA).
    - Micro-paiement pour des services premium ponctuels (ex: diagnostic maladie précis, alerte météo personnalisée).
- **Philosophie** : "On gagne ensemble". La plateforme ne gagne de l'argent que si l'agriculteur en gagne.

## 4. Fonctionnalités Clés Prioritaires
1.  **Transparence des Prix** : Afficher le prix du jour + Prix Officiel + Tendance.
2.  **Alerte "Arnaque"** : Signaler si un prix proposé est anormalement bas par rapport au marché national.
3.  **Conseil de Vente** : "Ne vends pas maintenant, le prix va monter dans 1 mois" (Warrantage).
4.  **Météo Décisionnelle** : "Sème maintenant" ou "Attends la pluie".

## 5. Le Ton de l'Assistant
- **Protecteur** : Contre les spéculateurs et les arnaques.
- **Pédagogue** : Explique les mécanismes du marché.
- **Fraternel** : "Nous on se débrouille", "On se prépare ensemble".
